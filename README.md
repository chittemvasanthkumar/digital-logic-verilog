# Digital Logic Design using Verilog HDL

This project demonstrates the design and verification of basic digital logic circuits using Verilog HDL.

## Modules Implemented
- 2:1 Multiplexer (Combinational logic)
- 4-bit Counter (Sequential logic)
- FSM-based Traffic Light Controller (Moore Machine)

## Tools Used
- Xilinx Vivado (Simulation)
- Verilog HDL

![Waveforms](https://github.com/chittemvasanthkumar/digital-logic-verilog/blob/main/waveforms/traffic_light_waveform.png?raw=true)

## Verification
All modules were verified using self-written testbenches and waveform analysis.

## Key Concepts
- RTL design
- Finite State Machines
- Timing behavior
